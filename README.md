# Cortinatges Esteba — Web

Web corporativa de Cortinatges Esteba. Next.js 15 + TypeScript + Tailwind + next-intl (ca/es/en/fr).

## Requisits

Node.js 20+ i npm 10+.

## Instal·lació i execució local

```bash
npm install && npm run dev
```

Obre [http://localhost:3000](http://localhost:3000) al navegador.
